# Panchganga Hydro Monitoring Network

Interactive hydro-meteorological monitoring dashboard
developed under the DST Research Project (2024–2027).

## Features
- Radar water level monitoring
- Rain gauge telemetry network
- Interactive GIS dashboard
- Site records and contacts
- Mobile responsive design
